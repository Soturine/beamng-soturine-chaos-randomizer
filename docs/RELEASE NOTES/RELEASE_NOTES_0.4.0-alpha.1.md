# Vehicle DNA and Persistence alpha

> Download `soturine_chaos_randomizer_0.4.0-alpha.1.zip` from this release's **Assets**. GitHub's automatically generated **Source code** archives are not installable BeamNG mod packages.

This prerelease adds bounded Vehicle DNA capture, a persistent Garage, strict and compatible preflight/restore modes, Replay Seed with generator version 4, schema-validated JSON import/export, and explicit environment fingerprints.

The attached `.sha256` and `release-manifest.json` belong to the attached installable ZIP. Verify the checksum after download.

Interactive BeamNG world/UI validation remains Pending. This artifact is an alpha for evidence gathering, not a gameplay-validated stable release. Restore Exact reports success only after strict read-back; targets whose slot tree cannot be inspected without changing the active vehicle remain `unverified` and are not restored as exact.
